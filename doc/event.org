* Event Management System
** To get the list of applications
#+BEGIN_SRC sql
  begin
     env.set_app_cd('EMS');
     em_code.APPLICATION_PK.get(o_applications => :o_applications);
  end;
  /
#+END_SRC

#+BEGIN_SRC sql
  begin
     env.set_app_cd('EMS');

  end;
  /
#+END_SRC
